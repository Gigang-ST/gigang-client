# 개인정보 보호법 - 친목단체 적용 요약

> 기강(스포츠 러닝 크루) 운영 시 참고용 법령 요약입니다.
> 법령 개정 시 내용이 달라질 수 있으므로 중요한 의사결정 전 최신 법령을 확인하세요.
> 기준 법령: 개인정보 보호법 (법률 제20897호, 2025년 10월 2일 시행)

---

## 1. 친목단체 특례 (제58조 제3항)

동창회·동호회 등 **순수 친목 도모 목적** 단체는 아래 3가지 의무가 **면제**됩니다.

| 면제 조항 | 내용 |
|----------|------|
| 제15조 | 개인정보 수집·이용 시 동의 요건 |
| 제30조 | 개인정보 처리방침 수립·공개 의무 |
| 제31조 | 개인정보 보호책임자 지정 의무 |

### 특례가 사라지는 경우
- 광고, 유료 서비스 등 **영리 활동**을 시작하는 경우
- 스폰서 브랜드에 **회원 정보를 제공**하는 경우 (제3자 제공 조항 적용)

---

## 2. 여전히 적용되는 조항

특례와 무관하게 아래는 반드시 지켜야 합니다.

| 조항 | 내용 | 기강 적용 |
|------|------|----------|
| 제17조 | 제3자 제공 시 동의 필요 | 스폰서에게 회원 정보 넘길 때 |
| 제18조 | 수집 목적 외 이용 금지 | 회원 정보를 모임 운영 외에 쓰면 안 됨 |
| 제24조 | 고유식별정보 처리 제한 | 주민번호·여권번호 수집 금지 |
| 제29조 | 안전성 확보 조치 | 계좌번호 등 암호화 권고 |

---

## 3. 수집 항목별 분류

| 항목 | 분류 | 비고 |
|------|------|------|
| 이름, 연락처, 생년, 성별 | 일반 개인정보 | 별도 규제 없음 |
| 계좌번호, 은행명 | 일반 개인정보 | 민감정보·고유식별정보 아님, 암호화 권고 |
| 프로필 사진 | 일반 개인정보 | 별도 규제 없음 |
| 주민등록번호 | 고유식별정보 | **수집 불가** |
| 건강정보, 생체정보 | 민감정보 | 별도 동의 필요 (수집 안 함) |

---

## 4. 동의 방식

현재 "계속 진행하면 동의로 간주" 방식은 엄밀히는 묵시적 동의입니다.
제15조가 면제되므로 법적 강제력은 낮지만, 향후 서비스 확장 시 체크박스 방식으로 전환 검토가 필요합니다.

---

## 5. 스폰서십 시나리오별 정리

| 시나리오 | 법적 문제 여부 |
|----------|--------------|
| 브랜드에서 기강에 물품/금전 제공 (회원 정보 공유 없음) | ✅ 문제없음 |
| 브랜드에 회원 이름·연락처 명단 제공 | ⚠️ 제17조 위반 — 회원 동의 필요 |
| 브랜드가 회원에게 직접 마케팅 문자 발송 | ⚠️ 제17조 + 정보통신망법 위반 |

---

## 6. 향후 서비스 확장 시 체크리스트

기강이 아래 방향으로 확장될 경우 법적 검토가 필요합니다.

- [ ] 유료 멤버십 도입
- [ ] 외부 광고 게재
- [ ] 회원 정보 기반 브랜드 제휴
- [ ] 해외 회원 수용 (GDPR 적용 가능)
- [ ] 건강 데이터(심박수, 혈압 등) 수집
