# 개인정보 처리 설계

> 📎 관련 법령 상세 내용은 [개인정보보호법 친목단체 적용 요약](./2026-03-26-개인정보보호법-친목단체-요약.md)을 참고하세요.

## 1. 법적 배경

기강은 스포츠 활동을 함께하는 **순수 친목 목적 단체**로, 개인정보 보호법 제58조 제3항에 따라 다음 조항이 적용 제외됩니다.

| 제외 조항 | 내용 |
|----------|------|
| 제15조 | 수집·이용 시 동의 요건 |
| 제30조 | 처리방침 수립·공개 의무 |
| 제31조 | 개인정보 보호책임자 지정 의무 |

> ⚠️ **주의**: 영리 활동(광고, 스폰서십, 유료 서비스 등)이 추가될 경우 위 특례가 사라지며 전면적인 법적 의무가 발생합니다.

나머지 조항(제17조 제3자 제공, 제18조 목적 외 이용 제한 등)은 그대로 적용됩니다.

## 2. 수집 항목 및 목적

### 필수 항목

| 항목 | 목적 |
|------|------|
| 이름 | 회원 식별, 예금주 확인 |
| 연락처 (전화번호) | 회원 식별, 모임 공지 |
| 생년월일 | 연령 확인 |
| 성별 | 기록 분류 |

### 선택 항목

| 항목 | 목적 | 비고 |
|------|------|------|
| 이메일 | 공지 전달 | OAuth 미제공 시 미수집 |
| 프로필 사진 | 회원 식별 | OAuth 자동 수집 |
| 은행명 | 회비 환급 | 입력 안 해도 가입 가능 |
| 계좌번호 | 회비 환급 | 입력 안 해도 가입 가능 |

### OAuth를 통해 자동 수집되는 항목

| 제공자 | 수집 항목 |
|--------|----------|
| 카카오 | 이름, 프로필 사진 |
| 구글 | 이름, 프로필 사진 |

## 3. 동의 방식

현재 로그인 페이지 하단에 "계속 진행하면 이용약관 및 개인정보처리방침에 동의한 것으로 간주합니다" 문구를 표시합니다.

친목단체 특례(제15조 면제)로 엄격한 동의 절차 의무는 없으나, 회원에게 수집 항목과 목적을 투명하게 알리는 것을 원칙으로 합니다.

## 5. 제3자 제공 및 위탁

- 현재 제3자 제공 없음
- 현재 외부 위탁 없음 (Supabase는 인프라 제공자로 처리 위탁에 해당할 수 있으나, 국내 친목단체 특례 범위 내에서 별도 고지 의무 없음)

### 국외 이전 (개인정보 보호법 제28조의8)

Supabase는 미국 기반 서비스로, 회원 데이터가 국외 서버에 저장됩니다.

| 항목 | 내용 |
|------|------|
| 이전 국가 | 미국 |
| 이전 업체 | Supabase Inc. |
| 이전 목적 | 데이터베이스 및 인증 인프라 운영 |
| 이전 항목 | 서비스 이용에 필요한 개인정보 전체 |
| 보유·이용 기간 | 회원 탈퇴 후 1년 |

> 친목단체 특례(제58조 제3항) 적용 대상이더라도 국외 이전은 별도로 고지하는 것을 원칙으로 합니다.

### 스폰서십 관련

외부 브랜드로부터 스폰서십(금전·물품)을 받을 수 있으나, 아래 원칙을 준수합니다.

| 상황 | 처리 방식 |
|------|----------|
| 브랜드가 기강에 금전·물품만 제공 | 회원 정보 공유 없음 → 별도 동의 불필요 |
| 브랜드에 회원 정보(이름, 연락처 등) 제공 시 | 제3자 제공에 해당 → **회원 사전 동의 필수** |

> 스폰서십을 받더라도 회원 정보를 외부에 넘기지 않는 한 친목단체 특례(제58조 제3항)는 유지됩니다.

## 6. 보안 조치

| 조치 | 내용 |
|------|------|
| 접근 제어 | Supabase RLS(Row Level Security)로 본인 데이터만 접근 가능 |
| 관리자 권한 | `admin = true` 회원만 전체 데이터 접근 |
| 계좌번호 | 평문 저장 중 → 추후 암호화 저장 검토 필요 |
| OAuth 토큰 | Supabase 내부에서 관리, 앱에 직접 노출되지 않음 |
