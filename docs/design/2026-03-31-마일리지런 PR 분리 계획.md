## 작업 방법

1. dev 에서 `feat/projects-pr{N}-{slug}` 브랜치 생성
2. 청사진 브랜치에서 해당 파일만 `git checkout feat/events-mileage-run -- <files>` 로 가져옴
3. 필요시 코드 수정/정리 후 커밋
4. dev 대상 PR 생성 (closes #{이슈번호} 포함)
5. 리뷰 후 squash merge
