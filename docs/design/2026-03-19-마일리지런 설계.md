# 마일리지런 프로젝트 기획서

> 기강 러닝크루 앱 — "프로젝트" 탭 첫 번째 프로젝트

## 1. 개요

기강 러닝크루 앱에 **프로젝트 탭**을 신규 추가하고, 첫 번째 프로젝트로 **마일리지런**을 운영한다.

- **프로젝트 탭**: 하단 탭바에 추가. 항상 활성 프로젝트 1개가 바로 표시됨.
- **마일리지런**: 5월~9월(5개월) 동안 크루원 각자 목표 마일리지를 설정하고, 매월 달성하며 점진적으로 목표를 높여가는 이벤트.
- 이 탭은 범용이며, 시즌에 따라 다른 프로젝트(예: 동계훈련)로 교체될 수 있음.

### 기술 참고

기존 앱: Next.js (App Router) + Supabase + Recharts + shadcn/ui + pnpm. 기존 패턴을 따를 것.

## 3. 마일리지 계산 규칙

### 3.1 종목별 기본 마일리지

| 종목 | 계산식 |
|------|--------|
| 러닝 | 거리(km) x 1 + 상승고도(m) / 100 |
| 트레일러닝 | 거리(km) x 1 + 상승고도(m) / 100 |
| 자전거 | 거리(km) / 4 + 상승고도(m) / 100 |
| 수영 | 거리(km) x 3 (상승고도 없음) |

- 거리 단위: km
- 상승고도 단위: m (수영 제외 전 종목 입력 가능)

## 4. 목표 및 월별 진행 규칙

### 4.1 초기 목표 설정

| 등급 | 시작 마일리지 |
|------|-------------|
| 초보 | 50 |
| 고수 | 100 |
| 초고수 | 자유 설정 |

### 4.2 월별 목표 갱신

목표는 정수값으로 관리한다.

| 당월 목표 범위 | 달성 시 다음달 | 미달성 시 다음달 |
|---------------|--------------|----------------|
| 0~49 | +10 | 유지 |
| 50~99 | +15 | 유지 |
| 100 이상 | +20 | 유지 |

- **달성 판정**: 해당 월 1일~말일 기록의 마일리지 합산 >= 해당 월 목표

### 4.3 목표 수정

- 해당 월 **1일~14일(포함)** 까지만 수정 가능. 15일부터 잠김.
- 현재 목표 **이상으로만** 수정 가능 (하향 불가)

## 5. 참여 신청 및 보증금

### 5.1 참여 신청

- 앱 내 "참여하기" 버튼으로 신청
- 신청 시 초기 목표 설정 (초보 50 / 고수 100 / 자유입력)
- 운영진이 보증금 입금 확인 후 참여 승인

### 5.2 보증금 구성

| 항목 | 금액 | 성격 |
|------|------|------|
| 월별 보증금 | 잔여 개월 수 x 1만원 | 환급 대상 |
| 참가비 | 1만원 | 환급 없음 → 회식비 |
| 싱글렛비 | 1만원 (미보유 시) | 환급 없음, 회식비 아님 → 별도 적립, 싱글렛 구매 시 반환 |

**예시**: 5월 시작 + 싱글렛 미보유 = 5만 + 1만 + 1만 = **7만원**

### 5.3 중도 참여

- 언제든 합류 가능
- 월 중 합류해도 해당 월부터 시작 (5월 10일 합류 → 5월부터)
- 중도 합류자도 합류 월의 전체 목표를 달성해야 함 (일할 계산 없음)
- 보증금은 잔여 개월분 기준 (합류 월 포함)

### 5.4 탈퇴/포기

- **중도 탈퇴 기능 없음** — 앱에서 탈퇴 버튼 없음
- 사실상 포기자 = 활동을 안 하는 사람. 별도 "포기" 처리 없이, 환급 계산에서 달성률 0%인 월은 자연스럽게 미환급 → 회식비 풀로 전환
- 환급은 9월 종료 시 일괄만, 예외 없음

## 6. 환급 계산

### 6.1 월별 환급

```
월 환급률 = min(해당 월 달성 마일리지 / 해당 월 목표, 1.0)
월 환급액 = 월 환급률 x 1만원
```

- 초과 달성해도 환급률은 최대 1.0 (100%)

### 6.2 총 환급액

```
총 환급액 = 각 참여 월의 환급액 합산
```

**예시**: 5개월 참여, 달성률 100%, 100%, 100%, 70%, 30%
→ (1.0 + 1.0 + 1.0 + 0.7 + 0.3) x 1만 = **4만원 환급**

### 6.3 환급 시점

- 9월 종료 시 일괄 환급

---

## 7. 회식비 계산

### 7.1 회식비 풀 구성

```
회식비 풀 = 전원의 (월별 보증금 - 환급액) 합산 + 전원 참가비 합
```

- 미환급 보증금에 포기자 보증금이 이미 포함됨 (달성률 0% → 전액 미환급)
- 싱글렛비는 풀에 포함되지 않음

### 7.2 1인당 회식비 상한 (참여 개월 수 비례 지분제)

**회식 참석자만** 대상으로 지분 계산:

```
1인 지분 = 본인 참여 개월 수
1인 상한 = 회식비 풀 x (본인 지분 / 참석자 전체 지분 합)
```

**예시**: 풀 50만원, A(5개월), B(5개월, 불참), C(3개월) 회식 참석
→ 참석자 지분: A(5) + C(3) = 8
→ A 상한: 50만 x 5/8 = 312,500원
→ C 상한: 50만 x 3/8 = 187,500원

### 7.3 규칙

- 상한 초과분은 자비 부담
- 회식 후 잔여금은 **크루 회비로 귀속**

## 8. 기록 입력

### 8.1 입력 항목

| 항목 | 타입 | 필수 | 비고 |
|------|------|------|------|
| 날짜 | Date | O | 기본값: 오늘 |
| 종목 | Select | O | 러닝 / 트레일러닝 / 자전거 / 수영 |
| 거리 | Number (km) | O | |
| 상승고도 | Number (m) | - | 수영 제외 전 종목, 선택 입력 |
| 이벤트 | Checkbox | - | 활성 이벤트 목록에서 복수 선택 |
| 후기 | Text | - | 한 줄 텍스트, 선택 |

### 8.2 입력 제한

- **미래 날짜**: 입력 불가
- **당월 기록**: 자유 입력
- **전월 기록**: 매월 3일까지만 입력 가능. 이후는 관리자만 가능.
  - 예: 6월 4일 → 5월 기록 입력 불가 (관리자 제외)

### 8.3 수정/삭제

- 본인 기록만 수정/삭제 가능
- 수정/삭제에도 입력과 동일한 시간 제한 적용 (전월 기록은 3일까지만, 이후 관리자만)

## 10. 내 현황 상세

### 10.1 표시 항목

- **당월 목표**: 현재 월의 목표 마일리지
- **현재 마일리지**: 당월 누적 마일리지
- **진행률**: 현재 마일리지 / 당월 목표 x 100 (%)
- **기간 대비 달성률**: `진행률 / (경과일 수 / 해당 월 총 일수)`. 경과일 = 오늘 날짜 - 1일 + 1. 예: 30일 중 15일 경과, 진행률 60% → 기간 대비 120%
- **일일 필요 거리**: (목표 - 현재 마일리지) / 잔여일 수 → km 환산 (러닝 기준, 1마일리지=1km). 목표 달성 시 "달성 완료" 표시.
- **환급 예정금액**: 현재까지 확정된 월 + 당월 예상 달성률 기반
- **회식비 지원 예상금액**: 현재 회식비 풀 기반 예상치

## 11. 전체 크루원 진행 현황 그래프

- **유형**: 시계열 누적 그래프 (Recharts 활용)
- **X축**: 날짜 (5월~9월)
- **Y축**: 누적 마일리지
- **라인**: 크루원별 1개 라인
- 월별 목표 달성 기준선 표시 (점선 등)

## 12. 데이터 모델 (참고용)

개발자가 기존 Supabase 패턴에 맞춰 설계하되, 필요한 엔티티:

- **project**: 프로젝트 정보 (이름, 기간, 상태)
- **project_participation**: 참여 정보 (회원, 프로젝트, 시작월, 보증금, 싱글렛 여부, 입금확인)
- **mileage_goal**: 월별 목표 (회원, 월, 목표값, 달성여부)
- **activity_log**: 기록 (회원, 날짜, 종목, 거리, 상승고도, 이벤트, 후기, 계산된 마일리지)
- **event_multiplier**: 이벤트 배율 (이름, 배율, 활성여부)
- **activity_log_event**: 기록-이벤트 연결 (다대다 관계, 또는 activity_log에 JSON 배열)

## 13. 기타 사항

- **시간대**: KST (UTC+9) 기준
- **마일리지 소수점**: 소수점 첫째자리까지 표시, 저장은 소수점 둘째자리까지
- **달성 판정**: 소수점 포함 전체 정밀도로 비교 (49.9 < 50 = 미달성)
- **후기 표시**: 최근 7일 내 후기 중 랜덤 3건 표출. 후기가 없으면 영역 미표시.
- **프로젝트 종료 후**: 9월 종료 후에도 기록/통계 열람 가능 (읽기 전용)

## 부록: 계산 예시

### 예시 1: 기본 기록

> 러닝 10km, 상승고도 200m, 비 올 때(x1.2)
> 기본: 10 + 2 = 12 마일리지
> 최종: 12 x 1.2 = **14.4 마일리지**

### 예시 2: 복합 이벤트

> 트레일러닝 15km, 상승고도 500m, 비(x1.2) + 모임(x1.1)
> 기본: 15 + 5 = 20 마일리지
> 최종: 20 x 1.2 x 1.1 = **26.4 마일리지**

### 예시 3: 환급 계산

> 5월 시작, 초보(50), 싱글렛 보유
> 납부: 5만(보증금) + 1만(참가비) = 6만원
> 결과: 5월 100% / 6월 100% / 7월 100% / 8월 80% / 9월 40%
> 환급: (1.0+1.0+1.0+0.8+0.4) x 1만 = **4.2만원**
> 미환급: 0.8만원 → 회식비 풀

### 예시 4: 회식비 배분

> 풀: 30만원
> 회식 참석자: A(5개월), B(4개월), C(3개월)
> 총 지분: 12
> A: 30만 x 5/12 = 125,000원
> B: 30만 x 4/12 = 100,000원
> C: 30만 x 3/12 = 75,000원
