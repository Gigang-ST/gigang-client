# /newbie 신규 가입 환영 페이지 설계

## 목적

신규 멤버에게 문자로 보내던 팀 소개 정보를 웹페이지로 제공하고, 자체 회원가입까지 자연스럽게 유도하는 랜딩 페이지.

## 라우트

- **경로**: `/newbie`
- **라우트 그룹**: `app/(info)/newbie/page.tsx` — BackHeader 레이아웃 사용
- **인증**: 불필요 (publicPaths에 추가)
- **서버 컴포넌트**: 정적 콘텐츠만 포함하므로 서버 컴포넌트로 구현
