# [v1.1.0] 모임 기능 및 푸시 알림 추가

> 게시 유형: 업데이트  
> 작성일: 2026-06-28

---

크루 **모임 일정**을 직접 만들고 모을 수 있게 되었고, 앱을 켜두지 않아도 새 소식을 받아보는 **푸시 알림**이 추가되었습니다.

## 모임 기능

대회 외에 정기런·번개런·이벤트 같은 크루 모임을 직접 등록하고 참석을 모을 수 있습니다.

- **모임 등록**: 홈 우하단 **+** 버튼 → **모임**에서 날짜·시간·장소·종목과 내용을 입력해 만듭니다. 정기/이벤트 구분과 러닝·트레일·하이록스·수영·사이클 종목 태그를 달 수 있습니다.
- **홈 캘린더 통합**: 등록한 모임이 대회·정보와 함께 홈 캘린더와 리스트뷰에 표시됩니다. 여러 날에 걸친 모임도 한눈에 보입니다.
- **참석 토글**: 모임 상세에서 참석 버튼 한 번으로 참석/취소가 되고, 참석자 목록이 바로 갱신됩니다.
- **댓글·공유**: 모임마다 댓글·답글로 대화할 수 있고, 공유하기 버튼으로 단톡방에 바로 알릴 수 있습니다. 등록 직후에는 상세 화면이 바로 열려 공유를 유도합니다.

## 푸시 알림

이제 앱을 열어두지 않아도 새 소식을 휴대폰 알림으로 받을 수 있습니다.

- 새 모임 등록, 참가 중인 모임의 수정·삭제, 새 정보 등록, 칭호 획득, 댓글·답글, 운영진 공지 등이 푸시로 도착합니다.
- 알림 내용 앞에 **[모임] · [정보]** 태그가 붙어 무슨 알림인지 한눈에 구분됩니다.
- 알림 종류별로 수신 여부를 켜고 끌 수 있습니다. (내 건의 답변은 필수 알림이라 항상 받습니다.)

> 푸시 알림은 **홈 화면에 설치한 앱(PWA)에서만** 동작합니다. 설치·설정 방법은 **사용 가이드**의 "푸시 알림 설정" 항목을 참고해 주세요.

---

**그 외 개선 및 버그 수정**

- 등록·기록 페이지 진입 속도를 개선해 깜빡임 없이 바로 표시됩니다.
- 캘린더 전환 속도를 개선했습니다.
- 댓글에서 본문·답글의 URL이 자동으로 링크로 표시됩니다.
- 24시간이 지난 댓글은 상대시간 대신 날짜(YY.MM.DD)로 표시됩니다.
- 댓글 작성 직후 본인 프로필이 기본 이미지로 보이던 문제를 수정했습니다.
- 모임 등록 후 참석 토글이 거꾸로 동작하던 문제와 캘린더 빈 칸 표시 문제를 수정했습니다.
- 홈 캘린더 색상·필터·비로그인 조회 동작을 개선했습니다.
- 폼 작성 중 다른 앱으로 전환해도 입력 내용이 유지되도록 개선했습니다.
