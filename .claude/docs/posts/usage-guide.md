## 0. 접속 방법 및 앱 설치 (2026.06.17)

### 카카오톡 등 다른 앱에서 링크로 접속 시 주의

카카오톡, 인스타그램 등 외부 앱에서 링크를 열면 해당 앱의 **인앱 브라우저**로 열리는데, 이 경우 **로그인이 정상적으로 동작하지 않을 수 있습니다.**

반드시 **Chrome 또는 Safari 등 일반 브라우저에서 직접 접속**해 주세요.

- **Android**: 링크를 길게 누르거나 우측 상단 메뉴 → "Chrome으로 열기" 선택
- **iOS**: 링크를 길게 누르거나 하단 공유 버튼 → "Safari로 열기" 선택

### 홈 화면에 추가 (앱처럼 사용하기)

브라우저 주소창에 매번 주소를 입력하는 번거로움 없이, 홈 화면에 아이콘을 추가하면 앱처럼 바로 실행할 수 있습니다.

**Chrome (Android)**
1. Chrome에서 기강 페이지 접속
2. 우측 상단 메뉴(⋮) → **"홈 화면에 추가"** 선택

**Safari (iOS)**
1. Safari에서 기강 페이지 접속
2. 하단 또는 상단 공유 버튼(□↑) → **"홈 화면에 추가"** 선택

## 1. 개인 설정 (2026.06.06)

### 프로필 수정 · 계좌 정보 · 회비 내역

[프로필] 탭 프로필 카드 바로 아래의 **내 정보 · 계좌 · 회비** 버튼을 탭하면 각 페이지로 바로 이동할 수 있습니다.

- **내 정보**: 이름, 프로필 사진 등 기본 정보 변경
- **계좌**: 회비 입금, 환급, 정산 등에 사용할 계좌 정보 등록 및 수정
- **회비**: 현재 잔액 및 납부 내역 확인

설정 화면(우측 상단 ⚙️)에서도 동일하게 접근할 수 있습니다.

### 디스플레이 모드 변경 (다크모드)

설정 화면 → **DISPLAY** 섹션의 **다크모드** 토글을 통해 화면 테마를 밝은 모드 / 어두운 모드로 전환할 수 있습니다. 별도로 설정하지 않으면 기기의 시스템 설정을 따릅니다.

## 2. 알림 설정 (2026.06.28)

새 모임·새 정보·댓글 등 새 소식을 알림으로 받을 수 있습니다. 앱 안에서 보는 **인앱 알림**(홈 우측 상단 🔔)은 기본으로 동작하고, 앱을 켜두지 않아도 받는 **휴대폰 푸시 알림**은 아래처럼 따로 켜야 합니다.

### 푸시 알림 켜기

> **푸시 알림은 홈 화면에 설치한 앱(PWA)에서만 동작합니다.** 브라우저로만 접속하면 푸시가 오지 않으니, 먼저 [0. 접속 방법 및 앱 설치](#0-접속-방법-및-앱-설치)를 참고해 홈 화면에 추가해 주세요. (iOS는 반드시 Safari, Android는 Chrome으로 설치)

**처음 켜는 방법**

홈 화면에 설치한 앱으로 처음 접속하면, 화면 상단에 **"🔔 기강이 빠졌군. 알림 켜라!"** 안내 배너가 나타납니다.

1. 배너의 **알림 켜기 🫡** 버튼을 탭합니다.
2. 휴대폰이 묻는 알림 권한 요청에 **허용**을 선택합니다.
3. "푸시 알림이 켜졌어요" 메시지가 뜨면 설정 완료입니다.

**처음에 안 켰거나 거부했다면**

배너를 닫거나 권한을 거부하면 **이 배너는 다시 자동으로 뜨지 않습니다.** 하지만 언제든 직접 다시 켤 수 있습니다.

1. 홈 화면 우측 상단 🔔 아이콘 탭
2. 팝업 우측 상단 ⚙️(설정) 탭
3. **푸시 알림** 토글을 ON으로 변경 → 권한 요청에 허용

> **휴대폰 설정에서 한 번 "거부"를 누른 경우**, 앱 안에서는 다시 켤 수 없습니다. 휴대폰 **설정 → 앱 → 기강(또는 브라우저) → 알림**에서 권한을 허용으로 바꾼 뒤 다시 시도해 주세요.

### 받을 수 있는 알림 종류

- **새 모임 등록**: 새 모임 일정이 올라오면 전체 멤버에게 발송됩니다.
- **참가 모임 수정·삭제**: 내가 참석한 모임이 수정·삭제되면 발송됩니다.
- **새 정보 등록**: 새 정보 공유 글이 올라오면 전체 멤버에게 발송됩니다.
- **칭호 획득**: 새 칭호가 부여되면 해당 멤버에게 발송됩니다.
- **댓글·답글**: 내 글이나 댓글에 댓글·답글이 달리면 발송됩니다.
- **운영진 공지**: 운영진이 직접 발송하는 알림입니다.
- **건의 답변**: 내가 남긴 건의에 운영진이 답하면 발송됩니다.

### 알림 종류별 수신 설정

받고 싶지 않은 알림은 종류별로 끌 수 있습니다.

1. 홈 화면 우측 상단 🔔 탭
2. 팝업 우측 상단 ⚙️(설정) 탭
3. 원하는 항목 토글 → ON / OFF
   - **새 모임 등록 / 참가 모임 수정·삭제 / 새 정보 등록 / 칭호 획득**을 개별로 끌 수 있습니다.
   - **건의 답변**은 필수 알림이라 끌 수 없습니다.

설정은 즉시 저장되며, 인앱 알림과 푸시 알림에 함께 적용됩니다.

### 알림이 안 와요

- 홈 화면에 **설치한 앱**으로 접속했는지 확인해 주세요. (브라우저 탭은 푸시 미지원)
- **푸시 알림** 토글이 ON인지 확인해 주세요.
- 받고 싶은 알림 종류가 켜져 있는지 위 **수신 설정**에서 확인해 주세요.
- Android는 절전(배터리 최적화) 때문에 알림이 지연될 수 있습니다.

## 3. 일정 (모임 · 대회 · 정보) (2026.06.28)

모임·대회·정보는 모두 홈의 **하나의 캘린더**에 함께 모이는 일정입니다. 보는 방법은 공통이고, 등록 방법만 종류별로 다릅니다.

### 캘린더 / 리스트 뷰 (공통)

홈 화면 상단에서 캘린더 뷰와 리스트 뷰를 전환할 수 있습니다.

- **캘린더 뷰**: 날짜를 탭하면 해당 날의 모임·대회·정보 목록이 캘린더 아래에 나타납니다. 목록에서 항목을 탭하면 상세 내용(참가자·참석자, 댓글 등)을 확인하고 참가·참석 신청할 수 있습니다.
- **리스트 뷰**: 월별로 전체 일정을 위아래로 스크롤하며 볼 수 있습니다. 항목을 탭하면 마찬가지로 상세 내용을 확인할 수 있습니다.

각 항목에는 종류(모임·대회·정보)와 종목 태그가 표시되어 한눈에 구분됩니다.

### 등록하기

홈 화면 우하단 **+** 버튼에서 **모임 · 대회 · 정보 공유**를 모두 등록할 수 있습니다.

### 모임

정기런·번개런·이벤트 같은 크루 모임을 직접 만들고 참석을 모을 수 있습니다.

**+** 버튼 → **모임**으로 등록합니다.

| 항목 | 설명 |
|------|------|
| 모임명 | 예) 양재천 정기런 |
| 날짜·시간 | 시작 일시 (여러 날 모임도 가능) |
| 장소 | 모이는 위치 |
| 종목 | 러닝 / 트레일 / 하이록스 / 수영 / 사이클 |
| 구분 | 정기 / 이벤트 |
| 내용 | 코스, 페이스, 준비물 등 안내 |

- 등록하면 모임 상세 화면이 바로 열리며, **공유하기 버튼으로 단톡방에 알려달라는 안내**가 표시됩니다.
- 상세 화면의 **참석** 버튼을 탭하면 참석으로 등록되고, 다시 탭하면 취소됩니다. 참석자 목록이 바로 갱신됩니다.

### 대회

**+** 버튼 → **대회**를 탭하면 날짜 기준으로 대회를 찾아 참가 신청할 수 있습니다. 목록에 없는 대회는 직접 추가할 수 있습니다.

> 기존 하단 탭의 대회 탭은 기강이야기 탭으로 교체되었습니다. 대회 전체 목록은 설정 > 대회 목록에서 확인할 수 있습니다.

**대회 등록 시 필요한 정보**

| 항목 | 예시 |
|------|------|
| 대회명 | 서울 마라톤 2025 |
| 종목 | 로드 러닝 / 트레일 러닝 / 울트라마라톤 / 철인3종 / 사이클 |
| 개최 장소 | 서울 잠실 올림픽공원 |
| 대회 날짜 | 2025년 04월 13일 |
| 공식 사이트 | 대회 홈페이지 URL (있는 경우) |

> **대회 정보는 정확하게 입력해 주세요.** 대회명, 장소, 날짜 등을 줄임말이나 약칭 없이 공식 명칭 그대로 적어주셔야 다른 멤버들이 검색하고 참가 신청할 때 혼선이 없습니다.

### 정보 공유

**+** 버튼 → **정보 공유**를 탭하면 대회 접수 일정, 러닝용품 세일, 세션 안내 등 팀 소식을 등록할 수 있습니다.

- 제목, 시작일시, 링크, 내용을 입력할 수 있습니다.
- 등록된 정보는 로그인하지 않아도 홈에서 확인할 수 있습니다.

### 댓글 · 공유하기 (공통)

모임·대회·정보 상세 화면에서는 모두 댓글·공유 기능을 쓸 수 있습니다.

- **댓글·답글**로 대화할 수 있으며, @이름으로 특정 멤버를 호출할 수 있습니다.
- 상세 화면 좌하단 **공유하기** 버튼을 탭하면 공유 옵션이 나타납니다.
  - **텍스트로 복사**: 제목·시간·내용·링크를 한 번에 클립보드에 복사합니다.
  - **다른 앱으로 공유**: 카카오톡, 문자 등 기기의 공유 기능으로 바로 보낼 수 있습니다.
- 공유 링크는 `gigang.kr/s/xxxxxx` 형태의 단축 URL로 생성됩니다. 링크를 받은 사람이 탭하면 해당 화면으로 바로 이동하며, 로그인하지 않아도 확인할 수 있습니다.

## 4. 회비 내역 확인 (2026.06.28)

[프로필] → **회비** 버튼 또는 설정 → **회비 내역**에서 확인할 수 있습니다.

### 현재 잔액

- 페이지 상단에 현재 잔액이 표시됩니다.
- 잔액 옆의 **기준일**은 관리자가 마지막으로 계산한 날짜입니다. 최근 납부가 반영되지 않았다면 아직 계산 전일 수 있습니다.
- 잔액이 마이너스(-)이면 미납 상태입니다.

### 납부 방법

- 페이지 상단의 **월 회비**와 **모임 계좌 정보**를 확인하세요.
- 계좌 이체 시 **입금자명을 정확한 본명**으로 입력해 주세요.

### 내역 확인

- **전체 / 회비 / 기타** 필터로 원하는 내역만 볼 수 있습니다.
- 날짜, 항목, 구분(입금·출금), 금액 순으로 표시됩니다.
- 최근 30건까지 표시됩니다.

### 참여 감면 현황

회비 페이지의 잔액 카드 아래에 **이번 달 참여 감면** 카드가 있습니다. 모임에 참여할수록 이번 달 회비가 얼마나 감면될지 확인할 수 있습니다.

- **① 정모 참여 또는 벙 개설**: 둘 중 하나만 충족하면 됩니다. (정모 1회 참여 *또는* 벙 1회 개설)
- **② 참여 횟수**: 진행 바에 4회 · 8회 지점과 각 단계 감면액이 표시됩니다.
- 카드 아래 **현재 감면 (마감 후 반영)** 에 지금까지 쌓인 예상 감면액이 보입니다.

> ①과 ②는 **둘 다 충족**해야 감면이 적용됩니다. 순서는 상관없어요. (예: 먼저 3번 참여하고 마지막에 벙을 1번 열어도 "참여 4회 + 벙 개설"로 조건이 채워집니다.)

자세한 감면 기준과 반영 시점은 **회비 감면 정책 공지**를 참고해 주세요.

### 문의하기

내역이 실제와 다르면, **문의하기** 버튼을 눌러 관리자에게 확인을 요청할 수 있습니다. 요청 전, **기준일자**를 확인해주세요.

## 5. 기록 입력 방법 (2026.06.17)

### 기록증으로 자동 입력 (OCR)

기록증 사진을 올리면 대회명·날짜·완주 시간이 자동으로 채워집니다.

1. [프로필] → **+ 기록 추가** 버튼 탭
2. **기록증 사진 업로드** 버튼을 눌러 사진 선택
3. AI가 내용을 읽어 입력 항목을 자동으로 채워줌
4. 대회명으로 검색된 목록에서 맞는 대회 선택 (없으면 직접 추가)
5. 코스와 시간을 확인한 뒤 등록

> AI가 읽은 내용이 실제와 다를 수 있으니 등록 전에 반드시 확인해 주세요.

### 직접 입력

[프로필] → PERSONAL BEST 섹션 우측 **+ 기록 추가** 버튼을 통해 참가한 대회와 완주 기록(시간, 순위 등)을 직접 입력할 수 있습니다.

### 과거 기록 수정 및 관리

[프로필] → PERSONAL BEST 섹션 우측 **기록 관리** 버튼에서 기존에 입력한 기록을 수정하거나 삭제할 수 있습니다.

과거 대회가 목록에 없다면, **기록 입력 화면에서 직접 대회를 추가**할 수 있습니다. 날짜를 선택하면 "대회 추가" 버튼이 활성화되며, 과거 날짜의 대회도 등록 가능합니다.

## 6. 칭호 설정 방법 (2026.06.03)

### 칭호 선택

[프로필] → 프로필 카드 우측의 **내 컬렉션** 버튼에서 보유 중인 칭호 목록을 확인하고, 프로필에 표시할 칭호를 선택할 수 있습니다.

### 칭호 획득 조건 확인

다른 멤버의 프로필에서 칭호를 탭하면 해당 칭호의 **획득 조건**을 확인할 수 있습니다. 아직 보유하지 못한 칭호도 조건을 미리 볼 수 있으니, 목표를 세우는 데 활용해 보세요.

## 7. 건의하기 (2026.06.17)

[프로필] 탭 바로가기 버튼의 **건의** 또는 설정 → ACCOUNT → **건의하기**에서 접근할 수 있습니다.

기강 활동이나 앱 이용 중 불편한 점, 개선 아이디어 등을 자유롭게 남겨주세요.

### 처리 현황 확인

건의하기 페이지 하단에서 내가 작성한 건의사항 목록과 처리 상태를 확인할 수 있습니다.

| 상태 | 의미 |
|------|------|
| 접수됨 | 건의가 접수된 상태 |
| 확인 중 | 운영진이 검토 중인 상태 |
| 처리 완료 | 반영 또는 처리된 상태 |
| 종결 | 검토 후 미반영으로 종결된 상태 |

운영진이 답변을 남기면 건의 항목 아래에 **운영진 답변**이 표시됩니다.
