## 0. 접속 방법 및 앱 설치

### 카카오톡 등 다른 앱에서 링크로 접속 시 주의

카카오톡, 인스타그램 등 외부 앱에서 링크를 열면 해당 앱의 **인앱 브라우저**로 열리는데, 이 경우 **로그인이 정상적으로 동작하지 않을 수 있습니다.**

반드시 **Chrome 또는 Safari 등 일반 브라우저에서 직접 접속**해 주세요.

- **Android**: 링크를 길게 누르거나 우측 상단 메뉴 → "Chrome으로 열기" 선택
- **iOS**: 링크를 길게 누르거나 하단 공유 버튼 → "Safari로 열기" 선택

### 홈 화면에 추가 (앱처럼 사용하기)

브라우저 주소창에 매번 주소를 입력하는 번거로움 없이, 홈 화면에 아이콘을 추가하면 앱처럼 바로 실행할 수 있습니다.

**Chrome (Android)**
1. Chrome에서 기강 페이지 접속
2. 우측 상단 메뉴(⋮) → **"홈 화면에 추가"** 선택

**Safari (iOS)**
1. Safari에서 기강 페이지 접속
2. 하단 또는 상단 공유 버튼(□↑) → **"홈 화면에 추가"** 선택

## 1. 개인 설정

### 프로필 수정 · 계좌 정보 · 회비 내역

[프로필] 탭 프로필 카드 바로 아래의 **내 정보 · 계좌 · 회비** 버튼을 탭하면 각 페이지로 바로 이동할 수 있습니다.

- **내 정보**: 이름, 프로필 사진 등 기본 정보 변경
- **계좌**: 회비 입금, 환급, 정산 등에 사용할 계좌 정보 등록 및 수정
- **회비**: 현재 잔액 및 납부·면제 내역 확인

설정 화면(우측 상단 ⚙️)에서도 동일하게 접근할 수 있습니다.

## 2. 회비 내역 확인

[프로필] → **회비** 버튼 또는 설정 → **회비 내역**에서 확인할 수 있습니다.

### 현재 잔액

- 페이지 상단에 현재 잔액이 표시됩니다.
- 잔액 옆의 **기준일**은 관리자가 마지막으로 계산한 날짜입니다. 최근 납부가 반영되지 않았다면 아직 계산 전일 수 있습니다.
- 잔액이 마이너스(-)이면 미납 상태입니다.

### 납부 방법

- 페이지 상단의 **월 회비**와 **모임 계좌 정보**를 확인하세요.
- 계좌 이체 시 **입금자명을 정확한 본명**으로 입력해 주세요.

### 내역 확인

- **전체 / 회비 / 기타** 필터로 원하는 내역만 볼 수 있습니다.
- 날짜, 항목, 구분(입금·출금·면제), 금액 순으로 표시됩니다.
- 최근 30건까지 표시됩니다.

### 문의하기

내액이 실제와 다르면, **문의하기** 버튼을 눌러 관리자에게 확인을 요청할 수 있습니다. 요청 전, **기준일자**를 확인해주세요.

---

### 디스플레이 모드 변경 (다크모드)

설정 화면 → **DISPLAY** 섹션의 **다크모드** 토글을 통해 화면 테마를 밝은 모드 / 어두운 모드로 전환할 수 있습니다.

## 3. 대회 등록 및 참가 방법

### 대회가 이미 등록된 경우

[대회] 탭에서 해당 대회를 찾아 참가 종목 선택 후 **참가 신청** 버튼을 누르면 됩니다.

### 대회가 등록되어 있지 않은 경우

대회에 참가하거나 기록을 남기려면 먼저 대회 정보를 등록해야 합니다. 관리자에게 요청하시거나 직접 등록할 수 있습니다.

**필요한 정보**

| 항목 | 예시 |
|------|------|
| 대회명 | 서울 마라톤 2025 |
| 종목 | 로드 러닝 / 트레일 러닝 / 울트라마라톤 / 철인3종 / 사이클 |
| 개최 장소 | 서울 잠실 올림픽공원 |
| 대회 날짜 | 2025년 04월 13일 |
| 공식 사이트 | 대회 홈페이지 URL (있는 경우) |

대회 정보 등록이 완료되면, 이후에는 해당 대회를 선택해 참가 신청이 가능합니다.
> **대회 정보는 정확하게 입력해 주세요.** 대회명, 장소, 날짜 등을 줄임말이나 약칭 없이 공식 명칭 그대로 적어주셔야 다른 멤버들이 검색하고 참가 신청할 때 혼선이 없습니다.

## 4. 기록 입력 방법

### 최근 대회 기록 입력

[프로필] → PERSONAL BEST 섹션 우측 **+ 기록 추가** 버튼을 통해 참가한 대회와 완주 기록(시간, 순위 등)을 입력할 수 있습니다.

### 과거 기록 수정 및 관리

[프로필] → PERSONAL BEST 섹션 우측 **기록 관리** 버튼에서 기존에 입력한 기록을 수정하거나 삭제할 수 있습니다.

과거 대회가 목록에 없다면, **기록 입력 화면에서 직접 대회를 추가**할 수 있습니다. 날짜를 선택하면 "대회 추가" 버튼이 활성화되며, 과거 날짜의 대회도 등록 가능합니다.

## 5. 칭호 설정 방법

### 칭호 선택

[프로필] → 프로필 카드 우측의 **내 컬렉션** 버튼에서 보유 중인 칭호 목록을 확인하고, 프로필에 표시할 칭호를 선택할 수 있습니다.

### 칭호 획득 조건 확인

다른 멤버의 프로필에서 칭호를 탭하면 해당 칭호의 **획득 조건**을 확인할 수 있습니다. 아직 보유하지 못한 칭호도 조건을 미리 볼 수 있으니, 목표를 세우는 데 활용해 보세요.
