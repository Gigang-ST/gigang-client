# 회비 관리 사용 가이드 (관리자)

> 최종 수정: 2026-06-06

---

## 전체 흐름

```
거래내역 업로드 → 회원 매칭 + 분류 → 확정 → 회비 계산 → 잔액/원장 확인
```

---

## 1. 업로드 탭

카카오뱅크 거래내역 엑셀 파일을 업로드합니다.

> ⚠️ **주의:** 카카오뱅크에서 다운로드한 파일은 기본적으로 비밀번호가 설정되어 있습니다.
> 업로드 전 반드시 Excel에서 파일을 열고 **다른 이름으로 저장**하여 비밀번호를 해제한 후 업로드해야 합니다.

- 업로드된 파일 목록과 처리 상태가 표시됩니다.
- 이미 업로드된 거래는 중복 적재되지 않습니다.
- 업로드 실수 시 **Rollback** 버튼으로 해당 업로드 건의 거래 전체를 삭제할 수 있습니다.
  - 단, 이미 확정된 거래가 포함된 경우 롤백이 불가합니다.

---

## 2. 거래내역 탭

업로드된 거래를 검토하고 확정합니다.

### 매칭

- 입금자 이름을 기반으로 회원이 자동 매칭됩니다.
- 매칭이 안 된 경우 직접 회원을 선택할 수 있습니다.
- 출금 거래도 필요시 회원과 매칭할 수 있습니다.

### 분류

- 각 거래의 항목(회비, 행사비 등)을 선택합니다.
- 항목 목록은 **회비 정책** 메뉴에서 관리합니다.

### 확정

- 회비(`due`) 거래는 회원 매칭이 필수입니다.
- 기타 거래(출금, 행사비 등)는 회원 매칭 없이도 확정 가능합니다.
- 체크박스로 여러 건을 선택해 일괄 확정할 수 있습니다.
- 확정된 거래는 수정할 수 없으며, 수정이 필요하면 **확정 취소** 후 재처리합니다.

> ⚠️ 이미 **회비 계산**에 반영된 거래는 확정 취소 전 해당 회원의 스냅샷을 **Rollback**해야 합니다.
> 순서: 회원별 잔액 탭에서 Rollback → 거래내역 탭에서 확정 취소 → 수정 후 재확정 → 회비 계산

---

## 3. 회비 계산

거래내역 탭 상단의 **회비 계산** 버튼을 누르면 실행됩니다.

- 마지막 계산 이후 새로 확정된 납부 내역과 면제 규칙을 반영하여 각 회원의 잔액을 갱신합니다.
- 면제 규칙이 설정된 회원은 이 시점에 면제 이력이 자동 생성됩니다.
- 특정 회원만 선택해서 계산하거나, 전체 회원을 한 번에 계산할 수 있습니다.
- 계산 후 잔액 옆의 **미반영** 표시가 사라지면 정상 반영된 것입니다.

---

## 4. 회원별 잔액 탭

전체 회원의 현재 잔액 스냅샷을 확인합니다.

- **미납 필터**로 잔액이 마이너스인 회원만 추려볼 수 있습니다.
- 마지막 계산 이후 새 확정 거래가 생긴 경우 **미반영** 뱃지가 표시됩니다. → 회비 계산을 다시 실행하세요.
- **Rollback** 버튼: 선택한 회원의 스냅샷을 이전 계산 시점으로 되돌립니다.
  - 잘못 확정된 거래를 취소해야 할 때 사용합니다.
  - Rollback 후에는 반드시 거래내역을 수정하고 회비 계산을 재실행해야 합니다.

---

## 5. 납부원장 탭

확정된 회비 납부 이력 전체를 조회합니다.

- **납부** 상태: 실제 반영된 납부 건
- **취소** 상태: 확정이 취소된 건 (잔액 계산에 포함되지 않음)

---

## 면제 관리 메뉴

회비 면제 규칙과 면제 이력을 관리합니다.

### 면제 규칙

- 특정 회원에게 기간(시작월 ~ 종료월)을 지정해 **전액 면제** 또는 **부분 면제**를 설정합니다.
- 규칙 자체는 즉시 반영되지 않으며, **회비 계산** 실행 시 해당 월의 면제 이력이 자동 생성됩니다.
- 규칙 수정/삭제 후 회비 계산을 다시 실행하면 변경사항이 반영됩니다.

### 면제 이력

- 실제 적용된 면제 내역입니다.
  - `규칙 적용`: 면제 규칙에 의해 회비 계산 시 자동 생성된 이력
  - `수동`: 관리자가 직접 추가한 이력
- 수동 면제 이력은 직접 추가/수정/삭제할 수 있습니다.
- 규칙으로 자동 생성된 이력은 Rollback 후 재계산으로 갱신합니다.

---

## 회비 문의 알림

회원이 회비 내역 페이지에서 **문의하기** 버튼을 누르면 owner에게 앱 알림이 발송됩니다.

- 알림 제목: **회비 문의**
- 알림 내용: `OOO님이 회비내역 확인을 요청하셨습니다.`
- 동일 회원이 하루에 1번만 요청 가능합니다.
- 알림 수신 후 해당 회원의 회비 내역을 관리자 페이지에서 확인하세요.

---

## 회비 정책 메뉴

월 회비 금액과 입금 항목 분류를 관리합니다.

### 회비 정책

- 기간별 월 회비 금액을 설정합니다.
- 여러 정책이 있을 경우 해당 날짜에 맞는 정책이 자동 적용됩니다.
- 정책 변경 후 **회비 계산**을 재실행해야 변경된 금액이 반영됩니다.

### 입금 항목

- 거래내역 분류 시 선택할 수 있는 항목 목록입니다 (회비, 행사비 등).
- 항목 추가/수정/삭제 및 순서 변경이 가능합니다.
