# [v1.0.0] 기록증 OCR 자동입력

> 게시 유형: 업데이트  
> 작성일: 2026-06-16

---

기록을 등록할 때 기록증 사진을 올리면 대회명·날짜·완주 시간이 자동으로 채워집니다.

**사용 방법**  
기록 추가 화면에서 기록증 사진을 업로드하면 AI가 내용을 읽어 입력 항목을 자동으로 채워줍니다.  
대회명으로 검색된 결과에서 맞는 대회를 선택하고, 코스와 시간을 확인한 뒤 등록하면 됩니다.  
목록에 없는 대회는 바로 추가할 수 있습니다.

**지원 종목**  
로드 러닝, 울트라마라톤, 트레일 러닝, 철인3종, 사이클 모두 인식됩니다.  
철인3종은 수영·바이크·런 구간 기록도 함께 추출됩니다.

**참고 사항**  
AI가 읽은 내용이 실제와 다를 수 있으니 등록 전에 반드시 확인해 주세요.  
JPG, PNG, HEIC 형식의 사진을 지원합니다.
