# [v1.0.0] 건의하기 기능 추가

> 게시 유형: 업데이트  
> 작성일: 2026-06-17

---

프로필 탭의 건의 기능이 추가되었습니다.

**건의하기**  
프로필 탭 바로가기 버튼과 설정 → ACCOUNT 항목에서 건의하기 페이지로 이동할 수 있습니다.  
기강 활동이나 앱 이용 중 불편한 점, 개선 아이디어 등을 자유롭게 남겨주세요.

**작성한 건의사항 목록**  
건의하기 페이지 하단에서 내가 작성한 건의사항 목록과 처리 현황을 확인할 수 있습니다.

| 상태 | 의미 |
|------|------|
| 접수됨 | 건의가 접수된 상태 |
| 확인 중 | 운영진이 검토 중인 상태 |
| 처리 완료 | 반영 또는 처리된 상태 |
| 종결 | 검토 후 미반영으로 종결된 상태 |

운영진이 답변을 남기면 건의 항목 아래에 운영진 답변이 표시됩니다.
