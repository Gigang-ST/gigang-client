# [v1.2.1] 프로필 수정 개선과 홈 속도 향상

> 게시 유형: 업데이트  
> 작성일: 2026-06-30

---

프로필 수정 화면이 더 안전하고 편리해졌고, 홈 화면과 캘린더가 한층 빨라졌습니다.

## 프로필 수정 개선

[프로필] → **내 정보** 화면이 개선되었습니다.

- 프로필 사진을 **기본 이미지로 되돌리기**가 가능해졌습니다. ("기본 이미지로" 버튼)
- 저장하면 **"저장했어요" 알림**으로 결과가 분명하게 표시됩니다.
- 프로필 사진을 아직 설정하지 않은 멤버에게 **고정된 랜덤 아바타**가 표시됩니다. 멤버마다 다른 그림이 일관되게 보여, 사진이 없어도 서로 구분하기 쉽습니다.

## 홈 화면 속도 향상

- 홈 화면이 **더 빠르게** 열립니다.
- 캘린더에서 **달을 넘길 때 훨씬 부드럽게** 전환됩니다.
- 대회 **등록·취소가 캘린더에 즉시** 반영됩니다. (응답을 기다리지 않고 바로 표시)

---

**그 외 개선**

- 마일리지(거리) 입력 시 **소수점 둘째 자리까지만** 입력되도록 정돈했습니다.
- 전반적인 **속도와 안정성**이 향상되었습니다.
