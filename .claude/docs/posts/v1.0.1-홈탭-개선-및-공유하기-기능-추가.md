# [v1.0.1] 홈탭 개선 및 공유하기 기능 추가

> 게시 유형: 업데이트  
> 작성일: 2026-06-17

---

홈 화면에 공유하기 기능이 추가되고, 속도와 편의성이 개선되었습니다.

**공유하기**  
정보 상세 화면 좌하단의 공유하기 버튼을 탭하면 두 가지 옵션이 나타납니다.  
**텍스트로 복사**는 제목·시간·내용·링크를 한 번에 클립보드에 복사하고, **다른 앱으로 공유**는 카카오톡·문자 등으로 바로 보낼 수 있습니다.  
공유 링크는 `gigang.kr/s/xxxxxx` 형태의 단축 URL로 생성되며, 링크를 받은 사람이 탭하면 해당 정보로 바로 이동합니다.


**홈탭 성능 최적화**  
- 홈 화면 진입 시 알림 목록을 미리 불러와 알림 패널이 즉시 표시됩니다.
- 알림을 탭하면 읽음 처리를 기다리지 않고 바로 해당 화면으로 이동합니다.
- 대회·정보를 탭했을 때 댓글을 미리 불러와 다이얼로그 열림이 빨라졌습니다.
- 월별 데이터 조회를 병렬 처리로 개선해 캘린더 전환 속도가 빨라졌습니다.

**댓글 optimistic UI**  
댓글·답글을 등록하면 서버 응답을 기다리지 않고 목록에 즉시 표시됩니다.  
전송에 실패한 경우 "전송 실패" 메시지가 표시됩니다.

---

**버그 수정**

- 리스트뷰에서 KST(한국 표준시) 기준으로 월이 올바르게 분류되지 않던 문제가 수정되었습니다.
- 다른 글로 전환 시 이전 댓글이 남아 있던 버그가 수정되었습니다.
- 정보 등록 후 리스트뷰가 즉시 갱신되지 않던 문제가 수정되었습니다.
